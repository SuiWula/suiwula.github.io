基于Hexo框架下NexT主题的网站优化

发表于 2017-08-11 | 分类于 网站搭建

本文承接本人上一篇文章，利用github和hexo搭建网站

更换主题

下载next主题

好的主题有很多，我们就以next主题为例来美化我们的博客吧！

在Blog根目录下，打开Git Bash,输入

git clone https://github.com/iissnan/hexo-theme-next themes/next

回车，git会自动帮你克隆最新的next主题到你的博客themes文件夹里。

安装主题

打开站点的_config.yml配置文件，修改主题为next

打开主题的_config.yml配置文件，不是站点主题文件，找到Scheme Settings

next主题有三个样式，我用的是Mist，你们可以自己试试看，选择你自己喜欢的样式（只需要把行首的#去除，#是注释），选择好后，再次部署网站，hexo g、hexo d，查看效果。选择其他主题，按照上述过程即可实现。

next主题优化

更改语言

打开站点配置文件，将language后面的内容修改为zh-Hans。注意,冒号之后必有空格!

如果出现汉字乱码的情况，下载sublime软件，用它打开配置文件进行修改。

然后在git里输入

hexo g

hexo d

将更改发布到自己的网站上。

设置右侧头像

编辑站点配置文件，添加如下内容

avatar: your avatar url

其中，your avatar url是完整的互联网url，你可以先将头像图片放在图床上，在复制链接，粘贴过来。

效果如下：

设置favicon图标

首先要有一个常见格式名(如.jpg, .png等)的图片作为备选favicon，选择一个favicon制作网站完成制作，例如比特虫是一个免费的在线制作ico图标网站。

将favicon.ico文件放在网站根目录下的source文件夹内即可。

用git刷新网站，就可以看到效果了。

添加背景图

首先找到一个背景图片放到 Blog（hexo工程文件）-> themes -> next -> source -> images 的路径下；

Blog（hexo工程文件）-> themes -> next -> source -> css -> _custom ，找到路径下的custom.styl文件，在文件的最上方加上一代码

body {  
 background:url(/images/backGround.jpg（这是你之前加的背景图片的名字）);  
     }

效果图：

如果想让背景图片不随着页面移动而移动，在上面的代码后加上一行代码

body {
  background:url(/images/background.jpeg);
  background-attachment: fixed;
}
